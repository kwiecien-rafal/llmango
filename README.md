# 🥭LLMango

Do large language models have a "favorite fruit", and does the answer change when you ask in Polish, Japanese, or Portuguese? **🥭LLMango** asks the same question across many languages and models, thousands of samples each, then aggregates the answers into distributions. The point isn't only how behavior shifts between languages, but how little genuine randomness there is behind it.
